# VaultKey 密码管理器 - 产品需求文档 (PRD)

## 1. 产品概述

VaultKey 是一款全平台密码管理器，对标 1Password，为用户提供安全、便捷的密码和敏感信息管理解决方案。支持网页端、iPhone、Mac 和浏览器插件，实现跨设备无缝同步。
- 解决用户密码记忆困难、重复使用弱密码、敏感信息管理混乱等安全痛点
- 面向注重数字安全的所有用户群体，从个人用户到家庭和小型团队

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 个人用户 | 邮箱注册 + 主密码 | 管理个人保管库，使用全部功能 |
| 家庭成员 | 邮箱注册 + 家庭邀请 | 共享家庭保管库，管理个人保管库 |
| 团队管理员 | 邮箱注册 + 团队创建 | 管理团队保管库、成员权限、安全策略 |
| 团队成员 | 邮箱注册 + 团队邀请 | 访问授权保管库，管理个人保管库 |

### 2.2 功能模块

1. **登录/解锁页**：主密码解锁、生物识别快捷入口、账户恢复
2. **仪表盘首页**：安全评分、最近使用、安全警报摘要、快捷操作
3. **项目列表页**：全类型项目浏览、搜索、筛选、排序、标签分类
4. **项目详情/编辑页**：查看/编辑密码、信用卡、身份、安全笔记等
5. **密码生成器**：可配置长度、字符集、可读性、批量生成
6. **安全中心（Watchtower）**：弱密码检测、重复密码、泄露监控、过期提醒
7. **保管库管理**：多保管库切换、创建/删除/归档、共享设置
8. **设置页**：账户管理、安全偏好、外观主题、导入导出、语言
9. **个人资料页**：头像、账户信息、订阅状态、活跃设备

## 3. 核心流程

### 3.1 用户首次使用流程
用户注册 → 设置主密码 → 保存恢复密钥 → 进入空保管库 → 引导添加第一个密码

### 3.2 日常使用流程
解锁保管库 → 搜索/浏览项目 → 查看详情/自动填充 → 修改/新增项目

### 3.3 安全检测流程
打开安全中心 → 查看安全评分 → 点击弱密码/泄露密码 → 跳转修改 → 使用密码生成器更新

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深邃藏蓝 (#0A1628) + 亮青色 (#00D4AA) 作为强调色，传达安全与科技感
- **辅助色**：暖白 (#F8F9FC)、浅灰 (#E2E8F0)、珊瑚红 (#FF6B6B 用于警告)
- **按钮风格**：圆角矩形 (8px)，主按钮采用渐变填充，次按钮为描边样式
- **字体**：标题使用 Plus Jakarta Sans (Web) / SF Pro Display (Apple)，正文使用 DM Sans (Web) / SF Pro Text (Apple)
- **布局风格**：左侧固定侧栏 + 右侧内容区，卡片式项目展示
- **图标风格**：线性图标 (24px)，2px 描边，圆角端点
- **动效**：页面切换淡入，卡片悬浮微浮起，密码复制成功勾选动画，侧栏展开收起流畅过渡

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 登录/解锁 | 主密码输入 | 居中卡片布局，VaultKey Logo + 输入框 + 渐变解锁按钮，深色背景带粒子动效 |
| 仪表盘 | 安全评分 | 左侧环形进度图，中心百分比数字，下方三色指标条，深色卡片容器 |
| 仪表盘 | 最近使用 | 横向滚动卡片，每卡含网站图标+标题+用户名，悬浮展开操作按钮 |
| 仪表盘 | 安全警报 | 红色/橙色警告卡片，左侧图标+右侧数字，点击可跳转 |
| 项目列表 | 分类侧栏 | 240px 固定宽度，图标+文字列表项，当前选中高亮背景 |
| 项目列表 | 搜索栏 | 顶部固定，带放大镜图标和快捷键提示 (⌘K) |
| 项目列表 | 项目卡片 | 双列网格布局，网站图标(48px)+标题+用户名+标签，悬浮显示操作按钮 |
| 项目详情 | 密码区域 | 密码以圆点显示，点击眼睛图标显示，复制按钮右侧，自动检测并显示TOTP |
| 密码生成器 | 配置面板 | 滑块控制长度，开关切换字符类型，实时预览区密码字母按类型着色 |
| 安全中心 | 监控列表 | 标签页切换(泄露/弱密码/重复/过期)，列表项带严重度标记和操作按钮 |

### 4.3 响应式设计

- **桌面端优先**：完整侧栏+内容区双栏布局
- **平板适配**：侧栏可折叠为图标模式，内容区自适应
- **手机适配**：侧栏变为底部标签栏，内容区全宽，项目卡片单列
- **触控优化**：按钮最小 44px 触控区域，滑动操作支持

### 4.4 浏览器插件设计

- **弹窗界面**：精简版项目列表，快速搜索，密码填充操作
- **自动填充提示**：检测到登录表单时，在输入框旁显示 VaultKey 图标
- **右键菜单**：右键可快速填充、生成密码、保存新登录
- **工具栏图标**：点击展开弹窗，显示锁定状态、快速访问

### 4.5 iPhone 端设计

- **Face ID 解锁**：启动即弹出 Face ID 验证
- **底部标签栏**：首页、项目、生成器、安全、设置
- **项目详情**：全屏卡片式布局，大号字体显示关键信息
- **自动填充集成**：通过 iOS AutoFill Provider 系统级集成

### 4.6 Mac 端设计

- **菜单栏快捷入口**：系统菜单栏图标，快速搜索和填充
- **主窗口**：与 Web 端一致的侧栏+内容区布局
- **Touch ID 解锁**：支持 Touch ID 快速解锁
- **全局快捷键**：⌘\ 快速填充，⌘⇧Space 快速搜索
